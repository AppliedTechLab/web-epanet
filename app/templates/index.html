Welcome to Web EPANET

Instructions


To use Web EPANET, follow these steps:

1. Locate the suitable .inp file from your hydraulic modeling software that you would like to use. This interface also accepts equivalent .txt files.


Example of suitable .inp file

2. If you would like to customize your certain variables, type them in the textboxes under the Customize section. Simply type in the new value (no spaces or letters), and press "Enter Variable." Note that some sections are mandatory for the program to function. These variables are labeled as "MANDATORY."

3. Upload your suitable file under the Upload section.

4. Then press the create button under the Create section and follow the download link to your new .inp file.
